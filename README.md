# Ft_printf
**To install:**
* `git clone https://github.com/jonghoon0421/Ft_printf.git`
* `make`

**To execute:**
* `ft_printf("format", arg1, arg2, ...)`
  * `Conversions`
    * `s, S, p, d, D, i, o, O, u, U, x, X, c, C`
  * `Flags`
    * `#, 0, -, +, *, and space`
  * `Length modifier`
  	* `hh, h, l, ll, j, z`
  * `Minimum field-width and precision`
  * `%% managed`
